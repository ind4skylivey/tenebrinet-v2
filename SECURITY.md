# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |
## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, email: **security@tenebrinet.io** (or your email)

You should receive a response within 48 hours. If not, follow up via GitHub DM.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Disclosure Policy

- Security issues are addressed within 7 days
- Fixes are released as patch versions
- Credit given to reporters (unless anonymity requested)
